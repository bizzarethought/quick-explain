# Changelog

All notable changes to QuickExplain will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-31

### Added
- Initial release of QuickExplain
- Multi-source knowledge integration:
  - Wikipedia for general summaries
  - Wikidata for structured facts
  - DBpedia for linked data abstracts
  - Dictionary API for word definitions
  - Wiktionary for etymology and language details
  - Open Library for book information
  - Numbers API for trivia and fun facts
- Automatic source selection with fallback
- Three tone options: Friendly, Academic, Power-User
- Dark/light theme with system preference detection
- Manual theme toggle
- Persistent user preferences via localStorage
- Smart popup positioning with viewport awareness
- Auto-hide after 8 seconds
- Selection limits (4 words, 60 characters)
- Settings panel for source and tone customization
- Keyboard support (Escape to close)
- Responsive design
- WCAG accessibility features
- Clean, modern UI with smooth transitions

### Technical
- Chrome Extension Manifest V3
- Vanilla JavaScript (no dependencies)
- CSS custom properties for theming
- RESTful API integrations
- Comprehensive error handling
- Code organization with clear sections
- Well-documented codebase

### Browser Support
- Chrome/Chromium browsers (v88+)
- Microsoft Edge (v88+)
- Opera (with Chromium engine)
- Brave browser
- Other Chromium-based browsers
