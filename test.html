QuickExplain Extension Test Page

How to test:
  1. Install the QuickExplain extension in your browser
  2. Highlight any of the words below (keep it to 4 words or less, 60 characters or less)
  3. A popup should appear with information about the highlighted text
  4. Click the ⚙️ settings button to change your source and tone preferences

Test with these examples:

Try highlighting: JavaScript, Wikipedia, artificial intelligence, or machine learning

Or try these longer phrases: web development, programming languages, software engineering

Note: The extension will only trigger for selections with 4 words or fewer and 60 characters or fewer to keep the popups concise.

Available Sources

  • Auto (Recommended): Automatically chooses the best source
  • Wikipedia: General background and explanations
  • Wikidata: Structured factual summaries
  • Dictionary: Word definitions and usage
  • Wiktionary: Etymology and language info
  • Open Library: Book and author details
  • Trivia: Fun facts and interesting details

Available Tones

  • More Friendly: Conversational and approachable
  • More Academic: Formal and precise
  • Power-User: Technical and detailed

Features

  • Smart source selection based on your preferences
  • Multiple tone options for different contexts
  • Dark/light theme support
  • Settings panel for customization
  • Auto-hide functionality
  • Responsive design

More Test Content

This is a test page for the QuickExplain browser extension. The extension allows you to quickly get information about highlighted text by showing a popup with relevant details from various knowledge sources.

Try highlighting different types of content:

  • Technical terms like API or JSON
  • Common words like computer or internet
  • Names like Albert Einstein or Leonardo da Vinci
  • Concepts like blockchain or cryptocurrency

The extension will fetch information from the selected knowledge source and display it in a clean, easy-to-read popup.
